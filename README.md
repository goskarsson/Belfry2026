# Belfry 2026 — Lifandi mælaborð

Read-only útgáfa af Belfry 2026 mælaborðinu, ætluð til að hýsa á GitHub Pages og deila með leikmönnum.

## Skrár

- `index.html` — Mælaborðið sjálft (eitt sjálfstætt skjal, engar ytri kröfur)
- `README.md` — Þessi skrá

## Uppsetning á GitHub Pages (eitt sinn, ~10 mín)

### 1. Búa til GitHub repo

1. Farðu á https://github.com/new
2. Repository name: `belfry-2026` (eða eitthvað sem þú vilt)
3. Visibility: **Public** (þarf að vera public til að GitHub Pages virki á free plani)
4. Hakaðu **við** "Add a README file"
5. Smelltu "Create repository"

### 2. Hlaða upp skjölunum

1. Smelltu á "Add file" → "Upload files"
2. Dragðu `index.html` og `README.md` úr þessari möppu inn
3. Skrunaðu niður og smelltu "Commit changes"

### 3. Kveikja á GitHub Pages

1. Farðu í repo Settings (tannhjólið efst)
2. Vinstra megin: smelltu "Pages"
3. Source: veldu **"Deploy from a branch"**
4. Branch: **main** og **/ (root)**
5. Smelltu "Save"
6. Bíddu í 1-2 mín — GitHub byggir síðuna

### 4. Sækja hlekkinn

Eftir 1-2 mín birtist hlekkur efst á Pages síðunni, t.d.:

```
https://goskarsson.github.io/belfry-2026/
```

Þetta er hlekkurinn sem þú deilir á Facebook hópinn.

## Dagleg uppfærsla (eftir hvern hring)

1. Þú slærð inn skor í Cowork-mælaborðinu eins og venjulega (eða ég les þau úr skjáskotum)
2. Þú segir mér: **"byggðu snapshot fyrir GitHub"**
3. Ég bý til nýtt `index.html` með núverandi stöðu innbökuðu
4. Þú ferð á GitHub.com → repo-ið → smellir á `index.html` → "Edit" táknið (pencil)
5. Eyðir öllu innihaldi og límir nýja útgáfu
6. Smellir "Commit changes"
7. Innan 30-60 sek uppfærist síðan á netinu

**Auðveldara verkflæði** (mælt með ef þú ert klár í GitHub Desktop):
1. Sækja GitHub Desktop frá https://desktop.github.com/
2. Klóna repo-ið á tölvuna
3. Þegar ég bý til nýtt `index.html`, dragðu það yfir gamla skjalið í möppunni
4. GitHub Desktop sér breytinguna → Commit + Push með einum smell
